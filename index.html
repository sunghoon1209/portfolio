SKILLS/TOOLS

HTML5

CSS

    미디어쿼리를 활용하여 반응형 디자인을, Flex 레이아웃으로 유연한 구조를, 애니메이션과 트랜지션으로 동적인 경험을 구현하며 크로스 브라우징을 고려

SASS

    변수, Nesting, Mixins 활용으로 가독성 높이고 중복 최소화하며, 웹 접근성 고려와 크로스 브라우징 보장

JAVASCRIPT

    기본 문법, 데이터 타입, DOM 조작, 이벤트 처리, 모듈화, 성능 최적화 등을 통해 자바스크립트를 효율적으로 사용

jQUERY

REACT

      React의 기본 개념과 생명주기를 이해, JSX 문법을 활용하여 컴포넌트를 구축하며, 상태 관리를 위해 Redux나 Context API 등의 도구를 활용, React Router와 같은 라이브러리를 사용하여 페이지 라우팅을 구현

Visual Studio Code

    플러그인을 활용한 확장성, 디버깅 기능을 활용한 버그 해결, 코드 자동 완성 및 리팩토링 기능을 활용한 코드 품짐 향상, 소스 제어 시스템과의 통합, 팀 협업을 위한 설정 및 확장 등을 숙달하여 개발 생산성 향상

Git

    기본적인 Git 명령어의 이해와 브랜치 관리, 충돌 해결, 명확하고 간결한 커밋 메시지 작성 및 Git의 원칙을 이해하여 효율적인 작업 흐름을 구축

GITHUB
