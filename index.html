HTML5

  • 웹 접근성 고려
  • 크로스 브라우징 이해

CSS

  • 미디어쿼리를 사용하여 다양한 기기와 화면크기에 대응하는 반응형 디자인 구현
  • Flex 레이아웃을 이용하여 유연하고 강력한 레이아웃 설계
  • 애니메이션과 트랜지션으로 사용자 경험을 향상시키고 동적인 웹페이지 구현
  • 크로스 브라우징 이해

SASS

  • 변수를 사용하여 가독성을 높이고 유지보수 용이
  • Nesting을 통해 가독성을 높이고 중복을 줄임
  • Mixins을 통해 코드의 중복을 줄이고 재사용성 향상
  • 웹 접근성 고려
  • 크로스 브라우징 이해

JAVASCRIPT

  • 기본 문법 및 데이터 타입 이해하고 활용
  • DOM을 이해하고 웹 페이지의 요소들을 선택하고 조작
  • 이벤트를 처리하고 이벤트 핸들러를 등록하여 사용자 상호작용에 반응
  • 코드를 모듈화하여 유지보수성을 높이고 재사용성 향상
  • 자바스크립트를 효율적으로 사용하여 웹 애플리케이션의 성능 최적화

jQUERY

  • JQuery를 이용하여 코드를 간소화하여 코드를 더 읽기 쉽고 유지보수하기 쉽게 할 수 있습니다.
